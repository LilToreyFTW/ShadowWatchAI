üéÆ Getting Started

Getting ShadowWatch AI running in your game project takes just a few minutes. Follow these steps to start building games with AI assistance.

  1. Download ShadowWatch AI Software
    Download the complete ShadowWatchAI-Software folder from the Download section below.
  2. Copy to Your Game Project
    Place the ShadowWatchAI-Software folder in the root directory of your game project.
  3. Install Dependencies
    Run npm install in the ShadowWatchAI-Software directory.
  4. Start the AI Server
    Run Start-ShadowWatchAI.bat (Windows) or node scripts/start-server.js.
  5. Open Control Panel
    Navigate to http://localhost:8080/cursor-control.html in your browser.
  6. Configure AI Provider
    Set up your preferred AI provider (Cursor AI or OpenAI) and start developing!
Quick Start Command:
cd ShadowWatchAI-Software && npm install && node scripts/start-server.js

üîß Troubleshooting

‚ùå Server Won't Start

# Check Node.js installation
node --version

# Install dependencies
cd ShadowWatchAI-Software
npm install

# Start server
node scripts/start-server.js

‚ùå AI Not Responding

  • Check internet connection
  • Verify API keys are configured (Cursor AI or OpenAI)
  • Check browser console for errors
  • Restart the server
  • Verify firewall settings

‚ùå "Protected Path" Errors

This is normal! ShadowWatch AI is correctly protecting its own codebase. The AI only works on your game project files.

  • Ensure your game files are in allowed directories (Assets/, Content/, src/, etc.)
  • Do not place game files inside ShadowWatchAI-Software/ folder
  • The protection system prevents accidental modification of AI files

‚ùå Engine Not Detected

  • Check project structure matches engine requirements
  • Ensure correct file extensions (.cs for Unity, .cpp for Unreal)
  • Run engine initialization in the control panel

‚ùå OpenAI Connection Issues

  • Verify API key starts with "sk-"
  • Check OpenAI account has sufficient credits
  • Test connection using the "Test" button
  • Try different GPT models if issues persist
Need Help? Check the logs in the ShadowWatchAI-Software directory for detailed error messages. All operations are logged for troubleshooting.
